# Simple CSV Dashboard (Streamlit)

A beginner-friendly Streamlit app that lets users upload a CSV, preview the data, apply a basic numeric filter, and generate simple charts.

## Features

- Upload any `.csv`
- Preview data in a table
- Filter rows by a numeric column using a slider
- Quick charts (bar + line)

## Sample Dataset

This project uses the Titanic dataset from Kaggle:

Source:
https://www.kaggle.com/c/titanic

The dataset is included for demonstration purposes only.

## Tech Stack

- Python
- Streamlit
- Pandas

## Setup (Local)

### 1) Create and activate a virtual environment

**macOS / Linux**

```bash
python3 -m venv .venv
source .venv/bin/activate
```
